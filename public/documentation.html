myFlix Movie API

The web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

API Endpoints
Business Logic URL1 HTTP Method Request body Response format Response body
Get a list of movies /movies GET None JSON Holding data about all the movies.
Get data about a single movie by title. /movies/[Title] GET None JSON Holding data about a single movie, containing
an description, genre, director, image URL and id.
{
"title": "Harry Potter",
"genre": "Fantasy",
"director": "Chris Columbus",
"description": "Harry Potter is a film series based on the
eponymous novels by author J. K. Rowling.",
"imageUrl": "https://upload.wikimedia.org/wikipedia/en/b/b5/Harry-film-logo.png",
"id": "b54537"
}
Get data about a genre by title. /movies/genres/[Title] GET None JSON Holding data about a single genre containing the description.
{
"genre": "Fantasy",
"description": "Fantasy stories can be about anything, anywhere,
anytime with essentially no limitations on what is possible."
}
Get data about a director by name. /movies/directors/[Name] GET None JSON Holding data about a director containing
an biography, birth year and death year.
{
"name": "Steven Spielberg",
"biography": "Steven Allan Spielberg is an American film director,
producer, and screenwriter.",
"birth year": "1946",
"death year": "None"
}

Add movies to users favorites list /users/[Username]/movies/[MovieID] POST None Message Indicating that a movie has been added.
Remove movies from users favorites list /users/[Username]/movies/[MovieID] DELETE None Message Indicating that a movie has been removed.
Deregister a existing user /users/[Username] DELETE None Message Indicating that a user has been successfully deregistered.

1For spaces use %20 Example: /movies/harry%20potter
